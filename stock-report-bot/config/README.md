# Config

초기 실행 설정은 `.env`를 사용합니다.

`theme_memory.csv`는 과거 Signal Evening ENEX 자료에서 추출한 종목-테마 사례 데이터입니다.
원본 ENEX 파일은 용량과 저작권/개인 자료 이슈가 있으므로 Git에 올리지 않고 `reference/enex/`에 로컬로만 보관합니다.

`stock_theme_keywords.csv`는 사용자가 정리한 종목별 대분류/중분류/키워드 사전입니다.
오늘 뉴스와 이 사전의 키워드가 같이 잡히면 해당 테마를 우선 분류하고, 뉴스가 애매하면 종목의 기본 섹터 fallback으로 사용합니다.

추후 확장할 수 있는 설정:

- 제외 종목
- 관심 테마 키워드
- 뉴스 검색 키워드
- 리포트 표시 종목 수 제한
- 텔레그램/이메일 알림 설정

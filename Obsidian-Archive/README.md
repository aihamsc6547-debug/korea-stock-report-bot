# Program Archive

이 폴더는 Obsidian에서 바로 열어 사용할 수 있는 프로젝트 아카이브입니다.

## 사용 방법

1. Obsidian에서 `Obsidian-Archive` 폴더를 Vault로 엽니다.
2. `00_Inbox/ChatGPT-share-source.md`에 공유 대화 내용을 붙여 넣습니다.
3. `01_Project/Program-Brief.md`를 기준으로 요구사항을 정리합니다.
4. 구현하면서 `03_Dev-Log/`와 `04_Decisions/`에 변경 기록과 의사결정을 남깁니다.

## 구조

- `00_Inbox`: 원문, 링크, 아이디어를 임시 보관
- `01_Project`: 제품/프로그램 개요와 요구사항
- `02_Design`: 기능 설계, 데이터 구조, 화면 흐름
- `03_Dev-Log`: 구현 로그와 테스트 기록
- `04_Decisions`: 중요한 선택과 이유
- `05_References`: 참고 링크와 자료
- `Templates`: 반복해서 쓸 문서 템플릿

## 공유 주의

일별 리포트에는 특정 날짜의 뉴스 링크와 요약이 들어갈 수 있습니다. 공개 저장소에 올릴 때는 생성된 리포트를 제외하고, 프로젝트 문서와 템플릿 중심으로 공유하는 것을 권장합니다.

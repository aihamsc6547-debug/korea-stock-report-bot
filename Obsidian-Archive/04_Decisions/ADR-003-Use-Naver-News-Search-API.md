# ADR-003: Use Naver News Search API

## 상태

Accepted

## 배경

종목이 당일 부각된 원인을 파악하려면 뉴스 제목, 요약, 링크가 필요하다. 기사 전문을 무단 크롤링하면 운영 안정성과 저작권 리스크가 생길 수 있다.

## 결정

초기 버전은 네이버 뉴스 검색 API의 검색 결과를 사용한다.

## 결과

- 공식 API 기반으로 검색 결과를 받을 수 있다.
- 기사 제목, 링크, 요약문, 발행일을 리포트에 붙일 수 있다.
- 기사 전문 요약은 하지 않고, API가 제공한 요약문과 제목 기반으로 부각 원인을 추정한다.

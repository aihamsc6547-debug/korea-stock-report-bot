# ADR-001: Use Obsidian Markdown

## 상태

Accepted

## 배경

사용자는 프로그램 제작 과정과 결과물을 Obsidian을 통해 아카이빙하고 싶어 한다.

## 결정

프로젝트 기록은 Markdown 파일 중심으로 관리한다. 각 문서는 Obsidian에서 바로 열 수 있도록 일반 폴더와 `.md` 파일로 구성한다.

## 결과

- 별도 데이터베이스 없이도 기록을 읽고 검색할 수 있다.
- 프로그램 요구사항, 설계, 구현 로그, 의사결정을 링크로 연결할 수 있다.
- 추후 실제 프로그램이 생성하는 결과도 같은 Vault에 저장할 수 있다.

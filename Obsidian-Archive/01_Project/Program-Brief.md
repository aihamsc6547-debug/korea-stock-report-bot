# 한국 주식 장마감 리포트 Bot

## 한 줄 설명

매일 한국 주식시장 장마감 후 급등/거래량 종목을 선별하고, 관련 뉴스와 부각 원인을 요약해 Obsidian Markdown 리포트로 저장한다.

## 만들고 싶은 것

- 프로그램 이름: Korea Stock Report Bot
- 핵심 사용자: 한국 주식 장마감 이후 특징주를 복기하고 싶은 개인 투자자/리서처
- 해결하려는 문제: 매일 흩어진 시세, 거래량, 뉴스 원인을 직접 찾아 정리하는 시간을 줄인다.
- 최종 산출물: `Market-Reports/YYYY/MM/YYYY-MM-DD-korea-market-report.md`

## 핵심 기능

- [x] 당일 상한가 추정 종목 선별
- [x] 당일 12% 이상 상승 종목 선별
- [x] 당일 거래량 1,000만 주 이상 종목 선별
- [x] 종목명, 종목코드, 시장, 상승률, 거래량, 거래대금 기록
- [x] 네이버 뉴스 검색 API 연동 구조
- [x] 기사 링크와 요약을 Markdown으로 렌더링
- [x] 뉴스 제목/요약 기반 부각 원인 추정
- [ ] 실제 API 키 입력 후 운영 데이터 검증
- [ ] macOS 자동 실행 등록

## 사용 흐름

1. 장마감 이후 스크립트가 실행된다.
2. KOSPI/KOSDAQ 전체 종목의 일별 시세를 가져온다.
3. 상승률/거래량 기준으로 종목을 선별한다.
4. 종목별 네이버 뉴스 검색 결과를 가져온다.
5. 기사 제목과 요약을 바탕으로 부각 원인을 추정한다.
6. Obsidian Vault 안에 일별 Markdown 리포트를 저장한다.
